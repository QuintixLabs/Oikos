Oikos

# About This Template

Oikos comes from the ancient Greek word for home or household. It is an open-source webring template that anyone can use and customize. You can rename it, change the design, add your own sites and members, and create your own webring.

The logo is a cube or square that looks like a little house and represents the idea of a home.

This template is built purely with HTML, CSS, and JavaScript. There are no special frameworks. The code is fully commented and organized so it is easy to understand and modify. The members are stored in a simple members.json file, making it easy to add or remove sites. You can also press "Use this template" on GitHub to start making your own customized webring.

Below you will see the "Joining" and other sections but this is just part of the template. You cannot actually join a webring here. It is just an example showing how a webring can look using Oikos.
